# Project Setup and Installation Steps

1. Create a Project Directory

2. Initialize a Node.js Project

    npm init -y
3. Install Cypress

    npm install cypress --save-dev

4. open the Cypress Test Runner GUI for the first time using the following command:

    npx cypress open
